📰 TextClip

Save text from any webpage with one click.
Organize with folders, summarize with your favorite AI.

Features

  • Drag → right-click → save (works on any webpage)
  • Folders, keyword extraction, full-text search
  • AI summary via clipboard — use any AI (ChatGPT, Gemini, Claude…)
  • All data stored locally — no servers, no tracking, no API keys

Privacy

TextClip stores all data locally in your Chrome profile. Nothing is sent to external servers.
